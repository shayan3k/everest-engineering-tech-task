# JavaScript Kiki's Courier CLI application

## A super fancy and safe CLI tool for Everest Engineering Tech Challenge.

# TO START USING APPLICATION, RUN

```
npm i --save/--dev-save
```

# RUN THE APPLICATION

```
npm start
```
